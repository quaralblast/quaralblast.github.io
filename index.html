Projects

Infant Fingerprint Scanner using an FPGA

I am currently working on a custom fingerprint sensor that will be able to capture infant biometric data. The scanner will use a deep learning model for the matching process, interfaced on an FPGA for real-time processing. This project is being overseen by Dr. Masadul Imtiaz at Clarkson University, and this project is a part of the NSF REU: Undergraduate Research in Sensor Development (Design, Manufacture, Analysis) and Implementation Pipeline (SDIP).

View Synthesis Across Cultures: Evaluating Model Performance in Diverse Visual Contexts

This experiment tested the generalizability of novel View Synthesis models when it came to diverse cultural contexts. We hypothesized that models would perform worse on scenes from non-eurocentric places. Instead, we found that scene complexity was likely a bigger hindrance to model performance. This project was apart of Brown's exploreCSR 2025 Social Responsible Artificial Intelligence program.
